# Quality Rubric

對任何極相林流程產出的 HTML / web 作品進行**系統協助判定**、輸出多維評分。屬客觀判斷、不代表創作意圖。

評分時機：用戶貼到 Claude Design / Vercel / GitHub / 任何平台**之前**。

---

## 評分維度（v0.1）

### 1. 協調性 · Cohesion

元素是否互相支撐、不互相打架。

| 分 | 判定 |
|:-:|---|
| 1-3 | 元素彼此脫節、視覺打架、無系統感 |
| 4-6 | 部分元素對齊、其他元素游離 |
| 7-9 | 全域元素互相支撐成系統 |
| 10 | 每個元素強化每個其他元素、無冗餘 |

### 2. 風格一致 · Style Consistency

Kit 語法是否從頭到尾守住、無美學漂移。

| 分 | 判定 |
|:-:|---|
| 1-3 | 多種美學擠在一起、register 漂移 |
| 4-6 | 主體一致、局部漂移 |
| 7-9 | Kit 語法全域守住 |
| 10 | Kit 語法守住 + 在語法內延伸出新意 |

### 3. 2D 視覺力 · 2D Visual Strength

字體層次 + 構圖節奏 + 色彩和諧。

| 分 | 判定 |
|:-:|---|
| 1-3 | 預設樣 · 無節奏 · 無層次 |
| 4-6 | 功能性可讀、缺性格 |
| 7-9 | 字體 / 構圖 / 色彩三軸都到位 |
| 10 | 純 2D 層獨立可作精品印刷品 |

### 4. 3D 視覺力 · 3D Visual Strength

空間深度 + 鏡頭意圖 + 材質本真。

| 分 | 判定 |
|:-:|---|
| 1-3 | 2D 平面假裝 3D · 無深度 |
| 4-6 | 3D 存在但扁、無大氣感 |
| 7-9 | 真深度 · 鏡頭服務敘事 · 材質有重量 |
| 10 | 達 cinematic 上限（Lusion / Villeneuve 級）|

> 純 2D 作品此維可標記 N/A、不計入綜合分。

### 5. 獨創性 · Originality

是否超出 Kit 預設、有 signature moment。

| 分 | 判定 |
|:-:|---|
| 1-3 | 模板感強、boilerplate |
| 4-6 | 可辨認 Kit、可預測 |
| 7-9 | 在 Kit 約束內出現意外、有 signature |
| 10 | 敘事 anchor 自己長出原創幾何 / motion / typography |

## 候補維度（v0.2 規劃）

| 維度 | 內容 |
|---|---|
| 6 · 敘事張力 · Narrative Tension | scroll 是否成故事弧、有起承轉合 |
| 7 · 互動深度 · Interaction Depth | 互動是否有意義、非裝飾 |
| 8 · 工藝完成度 · Craft Polish | 邊緣案例處理、字距 / 對齊 / 過場細節 |

## 評分流程（給 AI agent）

當用戶要求「幫我判定這個輸出」或拖入 HTML 時、agent 跑以下步驟：

1. **讀取輸出** · HTML 內容 + Kit 標記（若有）
2. **逐維打分** · 對 5 個維度各依上述判定條給 1-10 星
3. **計算綜合** · 取 5 維算術平均（小數一位）· 純 2D 作品取 4 維平均
4. **產生注記** · 對任何 < 8 維、給 1-2 條具體可改善線索
5. **輸出星卡** · 用上述格式輸出
6. **不評論** · agent 不額外評論「這個很棒」「我覺得」等主觀語、保持系統判定 framing

---

## 注意

- 評分為**判定參考**、非絕對判決。創作者保留所有最終決策權。
- 評分標準會隨 v0.2+ 版本演化、舊作品分數可能調整。
- 同一輸出在不同 Kit 標記下評分結果可能不同（Kit 約束改變判定基準）。
